## 一、檔案組織概覽

### 1.1 Mainframe 檔案類型

```
┌─────────────────────────────────────────────────────────────────┐
│              Mainframe 檔案類型分類                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  非 VSAM 檔案                                            │  │
│   │  ─────────────                                           │  │
│   │  • Sequential File (循序檔)                              │  │
│   │    - 依序存取，從頭到尾                                  │  │
│   │    - 適合批次處理                                        │  │
│   │                                                          │  │
│   │  • Indexed Sequential (索引循序)                         │  │
│   │    - 支援直接查詢                                        │  │
│   │    - 較少使用                                            │  │
│   │                                                          │  │
│   │  • Relative File (相對檔)                                │  │
│   │    - 以相對記錄號存取                                    │  │
│   │    - 較少使用                                            │  │
│   └─────────────────────────────────────────────────────────┘  │
│                              ↓                                  │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  VSAM 檔案 (Virtual Storage Access Method)               │  │
│   │  ─────────────────────────────────────────               │  │
│   │  • KSDS (Key-Sequenced Data Set)                         │  │
│   │    - 索引檔，以 Key 排序                                 │  │
│   │    - 最常用的 VSAM 類型                                  │  │
│   │                                                          │  │
│   │  • ESDS (Entry-Sequenced Data Set)                       │  │
│   │    - 循序檔，依進入順序                                  │  │
│   │    - 類似 Sequential File                                │  │
│   │                                                          │  │
│   │  • RRDS (Relative Record Data Set)                       │  │
│   │    - 相對記錄檔                                          │  │
│   │    - 以 RRN 直接存取                                     │  │
│   │                                                          │  │
│   │  • LDS (Linear Data Set)                                 │  │
│   │    - 線性資料集                                          │  │
│   │    - 特殊用途                                            │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 二、Sequential File

### 2.1 Sequential File 特性

```
┌─────────────────────────────────────────────────────────────────┐
│              Sequential File 特性                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   儲存方式：                                                     │
│   • 記錄依序儲存，一筆接一筆                                    │
│   • 無索引，無法直接跳到特定記錄                                │
│   • 新記錄只能加在檔案結尾                                      │
│                                                                 │
│   存取方式：                                                     │
│   • 只能循序讀取 (從頭到尾)                                     │
│   • 若要找第 N 筆，必須讀過前 N-1 筆                           │
│   • 寫入時只能加在結尾                                        │
│                                                                 │
│   ┌──────┬──────┬──────┬──────┬──────┐                        │
│   │ Rec1 │ Rec2 │ Rec3 │ Rec4 │ Rec5 │                        │
│   └──────┴──────┴──────┴──────┴──────┘                        │
│      ↑      ↑      ↑      ↑      ↑                            │
│   只能依序讀取，無法跳過                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 Sequential File 宣告

```cobol
       *---- Sequential File 宣告範例 -----------------------------
       
       ENVIRONMENT DIVISION.
       INPUT-OUTPUT SECTION.
       FILE-CONTROL.
           SELECT TRANSACTION-FILE
               ASSIGN TO 'TXNFILE'
               ORGANIZATION IS SEQUENTIAL
               ACCESS MODE IS SEQUENTIAL
               FILE STATUS IS WS-TXN-STATUS.
       
       DATA DIVISION.
       FILE SECTION.
       FD  TRANSACTION-FILE.
       01  TRANSACTION-RECORD.
           05  TXN-SEQ-NUM      PIC 9(10).
           05  TXN-DATE         PIC 9(8).
           05  TXN-AMOUNT       PIC S9(9)V99 COMP-3.
           05  TXN-DESCRIPTION  PIC X(40).
```

### 2.3 Sequential File 使用場景

| 場景 | 說明 | 範例 |
|------|------|------|
| **批次處理** | 日終批次作業 | 日終利息計算 |
| **報表產生** | 循序讀取產生報表 | 日報、月報 |
| **備份/還原** | 完整檔案複製 | 每日備份 |
| **資料交換** | 系統間檔案傳輸 | 匯入匯出檔 |
| **排序後處理** | 已排序檔案的後續處理 | 排序後的合併 |

## 三、VSAM 檔案

### 3.1 VSAM 概覽

VSAM (Virtual Storage Access Method) 是 IBM Mainframe 上的高效能檔案存取方法，提供比傳統檔案更好的效能和功能。

```
┌─────────────────────────────────────────────────────────────────┐
│              VSAM 檔案結構                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  Control Interval (CI) - 控制區間                        │  │
│   │  • 基本的儲存單位 (通常 4KB)                              │  │
│   │  • 包含多筆記錄                                          │  │
│   │  • 包含控制資訊 (記錄長度、空間管理等)                    │  │
│   └─────────────────────────────────────────────────────────┘  │
│                              ↓                                  │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  Control Area (CA) - 控制區域                            │  │
│   │  • 由多個 CI 組成 (通常 1MB)                              │  │
│   │  • 是空間分配的基本單位                                  │  │
│   └─────────────────────────────────────────────────────────┘  │
│                              ↓                                  │
│   ┌─────────────────────────────────────────────────────────┐  │
│   │  Index - 索引 (KSDS)                                     │  │
│   │  • 多層次索引結構                                        │  │
│   │  • 支援直接查詢                                          │  │
│   │  • 自動維護                                              │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
