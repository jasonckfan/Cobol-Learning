## 一、TSO/ISPF 基礎

### 1.1 TSO (Time Sharing Option)

TSO 是 Mainframe 上的互動式操作環境，讓使用者可以登入系統執行各種任務。

```
┌─────────────────────────────────────────────────────────────────┐
│              TSO 登入流程                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. 啟動終端機模擬器 (如 IBM Personal Communications)          │
│                                                                 │
│  2. 連線到 Mainframe                                            │
│     - 輸入連線位址                                              │
│     - 建立連線                                                  │
│                                                                 │
│  3. TSO 登入                                                    │
│     - 輸入 User ID                                              │
│     - 輸入 Password                                             │
│     - 選擇 PROC (通常按 Enter)                                  │
│                                                                 │
│  4. 進入 ISPF                                                   │
│     - 在 TSO READY 提示下輸入 ISPF                              │
│     - 進入 ISPF 主選單                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 ISPF (Interactive System Productivity Facility)

ISPF 是 Mainframe 上的圖形化操作介面，提供選單驅動的操作環境。

**常用選項**：

| 選項 | 名稱 | 用途 |
|------|------|------|
| **0** | Settings | 個人設定 |
| **1** | View | 檢視資料集 |
| **2** | Edit | 編輯資料集 |
| **3** | Utilities | 工具程式 |
| **3.4** | Data Set List | 資料集列表 |
| **4** | Foreground | 前景執行 |
| **5** | Batch | 批次提交 |
| **6** | Command | TSO 命令 |
| **7** | Dialog Test | 對話測試 |
| **M** | More | 更多選項 |

### 1.3 常用 ISPF 操作

```
瀏覽資料集 (Option 1 或 3.4):
─────────────────────────────
- 輸入資料集名稱: 'USERID.COBOL.SOURCE'
- 使用萬用字元: 'USERID.*'
- 檢視成員: 在成員前輸入 V
- 編輯成員: 在成員前輸入 E

編輯畫面常用指令:
────────────────
- F5 / RPFIND: 尋找下一個
- F6 / RCHANGE: 取代
- F12 / CANCEL: 取消
- SAVE: 儲存
- END / F3: 結束
- TOP: 跳到開頭
- BOTTOM: 跳到結尾

行指令:
───────
- I: 插入行
- D: 刪除行
- R: 複製行
- C / CC: 複製 (C=1行, CC=多行)
- M / MM: 移動
- A: 貼上到後面
- B: 貼上到前面
```

## 二、File-Aid

### 2.1 File-Aid 簡介

File-Aid 是強大的資料集瀏覽與編輯工具，特別適合檢視與修改 VSAM 檔案。

**主要功能**：
- 瀏覽各種格式檔案
- 以欄位層級檢視資料
- 搜尋與篩選資料
- 資料比較
- 資料編輯

### 2.2 File-Aid 基本操作

```
啟動 File-Aid:
──────────────
在 ISPF 命令行輸入: FILEAID

瀏覽檔案:
─────────
1. 選擇 Browse (選項 1)
2. 輸入檔案名稱
3. 選擇格式 (FORMatted / UNFormatted)

常用功能:
─────────
- FIND: 搜尋特定值
- FILTER: 設定篩選條件
- HEX: 切換十六進位顯示
- LEFT/RIGHT: 左右捲動
- PF7/PF8: 上下頁

範例 - 搜尋特定帳號:
────────────────────
FIND '123456789012' ACCT-NUMBER
```

## 三、版本控制工具

### 3.1 Endevor

Endevor 是 CA (現 Broadcom) 的軟體變更管理工具，用於控制原始碼版本與部署。

**主要功能**：
- 原始碼版本控制
- 變更追蹤
- 部署管理
- 權限控制

**基本流程**：
```
1. 取出 (Retrieve) 原始碼
2. 編輯修改
3. 新增 (Add) 變更
4. 生成 (Generate)
5. 移動 (Move) 到測試環境
6. 測試驗證
7. 移動 (Move) 到正式環境
```

### 3.2 Changeman

Changeman 是另一個常用的變更管理工具，功能類似 Endevor。

## 四、其他常用工具

### 4.1 SDSF (System Display and Search Facility)

SDSF 用於監控與管理批次作業。

**常用功能**：
- 查看作業佇列 (DA)
- 查看輸出佇列 (H)
- 查看日誌 (LOG)
- 查看系統資源 (RSYS)

```
SDSF 常用指令:
──────────────
- DA: 顯示作用中作業
- I: 顯示輸入佇列
- H: 顯示輸出佇列 (Held)
- O: 顯示輸出佇列 (Output)
- ST: 顯示作業狀態
- LOG: 顯示系統日誌

作業操作:
─────────
- S: 選擇作業查看詳情
- P: 清除作業
- ?: 顯示作業輸出
```

### 4.2 Coolgen / CA Gen

Coolgen (現 CA Gen) 是應用程式開發工具，支援模型驅動開發。

**主要功能**：
- 視覺化程式設計
- 自動程式碼生成
- 資料模型設計
- 畫面設計

### 4.3 Xpediter

Xpediter 是除錯工具，允許互動式追蹤程式執行。

**主要功能**：
- 設定中斷點
- 單步執行
- 檢視變數值
- 修改變數值
